This is a simple demo application showing the possibilities of Linked Open Data. Sadly enough, only the Dutch version actually uses linked data. This (internation) apllciation is based on the OpenStreetMap overpass-api and the turbo language.
Enter a location and click 'SEARCH'. The web application tries to find the location and show it on the map. The maps pans and zooms to show you the location. In a second step, the building structures are plotted as vectors on the map as well. In the last step: the application shows you the locations of the bars and pubs in the neightbourhood. Cheers!
